HTML and CSS FAQ

1. What do HTML and CSS stand for?

HTML stands for Hypertext Markup Language, and CSS stands for Cascading Style Sheets.

2. Would you rather use HTML or CSS, for putting paragraphs of text on a webpage?

I would use HTML for putting paragraphs of text on a webpage, as it is the primary markup language for creating the structure of a webpage.

3. Would you rather use HTML or CSS, to change the font and background color of a button?

I would use CSS to change the font and background color of a button, as CSS is used for styling and presentation.

4. What is an HTML tag?

An HTML tag is a code snippet that is used to define an element within an HTML document. It is enclosed in angle brackets and includes the name of the tag and any attributes.

5. What are the three parts of an HTML element?

The three parts of an HTML element are the opening tag, content, and closing tag.

6. What is the purpose of the doctype declaration?

The doctype declaration is used to specify the version of HTML used in the document and to provide instructions to web browsers on how to interpret the document.

7. What is the HTML element?

The HTML element is the root element of an HTML document and contains all other elements.

8. What is the purpose of the head element?

The head element is used to provide information about the document, such as the title, metadata, and links to external resources.

9. What is the purpose of the body element?

The body element is used to define the content of the document, such as text, images, and other media.

10. How do you create a paragraph in HTML?

To create a paragraph in HTML, use the <p> tag.

11. How do you create a heading in HTML?

To create a heading in HTML, use the <h1> to <h6> tags, with <h1> being the largest and <h6> being the smallest.

12. How many different levels of headings are there and what is the difference between them?

There are six different levels of headings in HTML, ranging from <h1> to <h6>. The difference between them is primarily in the size of the text and the degree of emphasis.

13. What element should you use to make text bold and important?

To make text bold and important, use the <strong> tag.

14. What element should you use to make text italicized to add emphasis to it?

You should use the <em> element to make text italicized to add emphasis to it.

15. What relationship does an element have with any nested elements within it?

An element with nested elements within it is called the parent element, and the nested elements are called the child elements. The parent element contains the child elements.

16. What relationship do two elements have if they are at the same level of nesting?

If two elements are at the same level of nesting, they are called sibling elements.

17. How do you create HTML comments?

You can create an HTML comment using the <!-- and --> tags. Anything in between these tags will be considered as a comment and will not be displayed on the webpage.

18. What HTML tag is used to create an unordered list?

You should use the <ul> element to create an unordered list.

19. What HTML tag is used to create an ordered list?

You should use the <ol> element to create an ordered list.

20. What HTML tag is used to create list items within both unordered and ordered lists?

You should use the <li> element to create list items within both unordered and ordered lists.

21. What element is used to create a link?

You should use the <a> element to create a link.

22. What is an attribute?

An attribute provides additional information about an element and is always specified in the start tag of the element using the attribute name and value. For example, in the <a> element, the attribute href is used to specify the URL that the link should go to.

23. What attribute tells links where to go?

The href attribute tells links where to go. It is used to specify the URL of the page or file that the link should go to.

24. What is the difference between an absolute and relative link?

An absolute link specifies the complete URL of the target page, including the protocol (e.g. http or https), domain name, and path to the page. A relative link, on the other hand, specifies the path to the target page relative to the location of the current page. Relative links are often shorter and more convenient to use, especially when linking to pages within the same website.

25. Which element is used to display an image?

The <img> element is used to display an image in HTML. It is a self-closing tag and requires a few attributes to function properly, such as the src attribute, which specifies the URL of the image file, and the alt attribute, which provides alternative text for the image.

26. What two attributes do images always need to have?

Images in HTML always need to have the src and alt attributes. The src attribute specifies the URL of the image file, while the alt attribute provides alternative text for the image. The alt attribute is important for accessibility reasons, as it allows screen readers and other assistive technologies to describe the image to users who cannot see it.

27. How do you access a parent directory in a file path?

To access a parent directory in a file path, you can use the .. notation. This notation represents the parent directory of the current directory. For example, if you have a file located at http://example.com/folder1/folder2/file.html and you want to link to a file in folder1, you can use a relative link like this: <a href="../index.html">Go to homepage</a>.

28. What are the four main image formats that you can use for images on the web?

The four main image formats that you can use for images on the web are:

  • JPEG: A lossy format that is best for photographs and images with lots of color gradients.
  • PNG: A lossless format that supports transparency and is best for images with sharp lines and text.
  • GIF: A format that supports animation and transparency, but has a limited color palette and is best for simple graphics and animations.
  • SVG: A vector format that scales well and supports interactivity, but may not be supported by all browsers.
